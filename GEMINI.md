# Project Goal: Web Application Development

This project focuses on the development of a modern, visually appealing, and functional web application.

## Core Mandates
- **Visual Excellence:** The application must feel "alive" and polished, with a strong emphasis on consistent spacing, typography, and interactive feedback.
- **Tech Stack:** Unless otherwise specified, the default stack is **React (TypeScript)** with **Vanilla CSS** for the frontend, and **Node.js (Express)** or **Python (FastAPI)** for the backend.
- **Styling:** Prefer Vanilla CSS for maximum flexibility. Avoid TailwindCSS unless explicitly requested.
- **Prototypes:** Focus on building substantially complete prototypes that provide a rich user experience from the start.

## Development Workflow
All feature implementations and architectural changes must follow the Research -> Strategy -> Execution lifecycle defined in the system instructions.
